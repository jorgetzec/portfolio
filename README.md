## About

A minimalist, code-inspired portfolio built with React, TypeScript, and TailwindCSS. Features include an animated DNA helix in the hero section, project categorization, and interactive project detail modals.

## Tech Stack

- React 19 + TypeScript
- Vite
- TailwindCSS
- Framer Motion
- React Hook Form

npm install
